Bước 7: Configure VM and setup local(linux)
      + download anaconda: 
            wget https://repo.anaconda.com/archive/Anaconda3-2024.02-1-Linux-x86_64.sh
            chạy anaconda dùng lệnh : bash anaconda(tab) và enter để tải packet
      + mở gitbash khác(root)
            gõ lệnh cd .ssh/ để vào terminal
            tạo file config: touch config 
            code config để vào vs code 
                  Host {Tên VM}
                  HostName {Địa chỉ External IP}
                  User {Tên user}
                  IdentityFile {path chứa file gcp}
            Ví dụ:  Host de-zoomcamp
                    HostName 34.80.141.97
                    User minhnghia
                    IdentityFile c:/Users/ADMIN/.ssh/gcp
      + cd về root
      + gõ lệnh ssh de-zoomcamp để vào máy chủ cloud
      + Trong máy chủ(de-zoomcamp) 
            gõ lệnh less .bashrc để xem anaconda đã được cài đặt chưa và nhấn q để thoát ra
            nếu chưa cài đặt thì thêm anaconda vào đường dẫn (path enviromen): export PATH="/home/minhnghia/anaconda3/bin:$PATH" sau đó gõ lệnh conda init
            gõ lệnh source .bashrc để về base
            +down load docker: sudo apt-get update
                               sudo apt-get install docker.io
                               sudo groupadd docker
                               sudo gpasswd -a $USER docker
                               sudo service docker restart
                               logout
                               ssh de-zoomcamp
            +down docker compose: link git: https://github.com/docker/compose/releases
                              mkdir bin
                              cd bin/
                              wget https://github.com/docker/compose/releases/download/v2.25.0/docker-compose-linux-x86_64 -O docker-compose
                              chmod +x docker-compose
            +add thư mục bin trong đường dẫn toàn cục
                              gõ lệnh nano .bashrc dể edit file
                              thêm export PATH="${HOME}/bin:${PATH}" sau đó ctrl o -> enter -> ctrl x

            +Clone project git của mình về

      +open file và chọn folder trên remote
